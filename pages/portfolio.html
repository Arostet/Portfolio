AcuTrack: The Acupuncture Treatment Tracker
Github Demo

This full-stack application, built with a Node.js backend and a PostgreSQL relational database, offers a comprehensive solution for acupuncture treatment tracking. It features a React with Redux frontend, elegantly designed using Material UI. The app excels in data analysis, user and client management, streamlining the process of tracking and analyzing acupuncture treatments for both practitioners and clients.
